Ce projet est une application de ChatBot interactive construite avec Node.js pour le backend et une interface moderne en HTML/CSS/JS. Il exploite la puissance de l'intelligence artificielle via l'infrastructure Groq, utilisant le modèle Llama 3 pour fournir des réponses instantanées et précises.

🚀 Fonctionnalités
Réponses en temps réel : Une communication fluide entre l'utilisateur et l'IA.

Performance extrême : Intégration avec Groq Cloud pour une génération de texte ultra-rapide.

Interface Intuitive : Design épuré et responsive pour une expérience utilisateur optimale.

Sécurité des données : Gestion des clés d'API via des variables d'environnement (.env).

🛠️ Technologies utilisées
Frontend : HTML5, CSS3, JavaScript (Fetch API).

Backend : Node.js, Express.js.

Moteur IA : SDK Groq (Modèle Llama 3.3).

Gestion de packages : NPM (Dotenv, Cors, Express).

📂 Structure du Projet
index.js : Le serveur Express qui gère la logique de l'application et les appels API.

index.html : L'interface utilisateur du chatbot.

.env : Fichier de configuration pour stocker la clé API en toute sécurité (non inclus sur GitHub).

package.json : Liste des dépendances et scripts du projet.
